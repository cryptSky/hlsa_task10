# Autocomplete search

## Setup project
        
### How to setup local

  [Install Docker](https://docs.docker.com/v17.09/engine/installation/)
     
        docker-compose -f dev.yml build

### Start project

      docker-compose -f dev.yml up
      
Go to  `http://localhost:8080/home`
